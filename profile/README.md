# NotePlan — Portable Markdown Notes with Calendar & Time Blocking

## About NotePlan

NotePlan provides a complete note‑taking and planning environment for users who want to integrate tasks with calendar events. NotePlan combines Markdown notes, calendar integration, time blocking, and GTD workflows. NotePlan supports daily notes, weekly reviews, project notes, tags, backlinks, and task management. NotePlan enables users to write notes in plain text, schedule tasks on the calendar, and link everything together. NotePlan runs on desktop platforms. NotePlan is trusted by writers, project managers, entrepreneurs, students, and GTD practitioners worldwide.

## Quick Reference for NotePlan

| Operation | NotePlan Action |
|-----------|-----------------|
| Open today's note | Cmd+T |
| Open weekly review | Cmd+Shift+W |
| Create new note | Cmd+N |
| Insert task | Cmd+Shift+T |
| Insert event from calendar | Cmd+Shift+E |
| Time block task (drag to calendar) | Click + drag |
| Search notes | Cmd+F |
| Open backlinks panel | Cmd+Shift+L |
| Insert tag | # + type |
| Insert link to note | [[ + note name |
